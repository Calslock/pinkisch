# pinkisch
 
Theme / Configuration files for Conky 1.10+
 
![](screenshot.png)

## Installation

Copy files to ~/.conky/pinkisch/. Run conky.sh to start widgets.

In case of a different theme catalog change paths in `conky.sh` and path to lua file in `conky3.sh`.
